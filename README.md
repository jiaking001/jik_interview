# JIK-面试刷题平台

## 项目介绍

​	基于 **Nunu** + **Redis** + **Elasticsearch** 的面试刷题平台；管理员可创建并批量管理题目；用户可 **分词检索** 题目，在线刷题并查看刷题日历；

## 技术选型

- Nunu 框架

- MySQL 数据库 + GORM
- Elasticsearch 搜索引擎
- Sa-Token 权限控制
- HotKey 热点探测
- Sentinel 流量控制
- Nacos 配置中心
- 多角度项目优化：性能、安全性、可用性

## 模块介绍

### 用户模块

登录，注册，注销，获取用户登录状态，签到（ BitMap 实现），获取签到记录

（管理员）查看用户列表，添加用户，删除用户，修改用户

### 题库模块

查看题库列表

（管理员）查看题库列表，添加题库，删除题库，修改题库

### 题目模块

查看题目列表

（管理员）查看题目列表，添加题目，删除题目，修改题目
